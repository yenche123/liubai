# Liubai


## 概览 📂

```
.
├─ liubai-backends
│  └─ liubai-laf
├─ liubai-docs
└─ liubai-frontends
   └─ liubai-web
```

`liubai-web` 为前端根目录，在该目录下运行 `pnpm i` 和 `pnpm dev` 即可启动留白前端。

`liubai-laf` 为后端根目录，使用 [Laf](https://github.com/labring/laf) 构建；该目录下的 `cloud-functions/` 存放本项目的云函数。

`liubai-docs` 为文档根目录，使用 [VitePress](https://vitepress.dev/) 构建。

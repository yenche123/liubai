# 设置

## 关于导出

### `liubai-markdown.zip` 文件结构:

由以下一个个 `/YYYY-MM-DD hh:mm:ss` 文件夹构成一个 `liubai-markdown.zip`
```
--- /YYYY-MM-DD hh:mm:ss
 |
 |--- README.md
 |
 |--- /assets
   |
   |--- 1.jpg
   |--- 2.jpg
   |--- name.文件名后缀
```


### `liubai-json.zip` 文件结构:

由以下一个个 `/YYYY-MM-DD hh:mm:ss` 文件夹构成一个 `liubai-json.zip`
```
--- /YYYY-MM-DD hh:mm:ss
 |
 |--- card.json
 |
 |--- /assets
   |
   |--- ${id1}.jpg
   |--- ${id2}.jpg
   |--- ${id3}.文件名后缀
```
